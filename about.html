Minibus Routes

About Us

The purpose of establishing this web page is to provide an interface page for those who want to access route and various attribute data related to minibus routes. Thus, the user will have the opportunity to examine the location information of the minibus routes on the map and can reach the route name and departure time of the relevant minibus. In addition, a web page with route identification and deletion possibilities for users approved by the system has been designed and an up-to-date and sustainable information screen has been created.


Attributes to be presented for minibus routes on the map:
  • Locations of the routes
  • Route Name
  • Departure Time

    • On the page of adding the route information of the minibus lines, four locations can be determined on the map and transferred to a destination database between the relevant locations. When it is desired to delete the route information, all route names can be listed and deleted from the desired route database via the "Delete Route" button.
